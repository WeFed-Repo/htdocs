Elenco Disposizioni

In questa pagina vengono elencate tutte le disposizioni preparate, suddivise per azienda, tipologia di operazione e quantità.

Sono presenti i seguenti dati:

Azienda: azienda per la quale è stata creata la disposizione;
Tipo Dispositivo: viene specificata la natura del flusso dispositivo;
N. Disposizioni: viene riportato il numero delle disposizioni suddivise per tipologia;
Tot. Disposizioni: viene riportato il totale di tutte le disposizioni per ogni azienda.

Cliccando sul relativo link presente nella colonna "N. Disposizioni" si accede alla pagina dove è possibile aggregare le disposizioni per la creazione delle distinte. Le distinte così create dovranno poi essere autorizzate per l'invio alla Banca.
N.B. L'operazone di aggregazione delle disposizioni è necessaria anche per le disposizioni singole.

Nella pagina è inoltre possibile modificare o eliminare le disposizioni inserite.
