NOTE

Le immaigni SVG del sito vengono trasformate in SVG-inline all'onready della pagina.
(viene trasformata di default qualsiasi immagine che possiede un path che termina con .svg)

Nell'esempio di seguito è applicata all'icona la classe CSS ".icon-style-demo" che colora di rosso l'icona.

Se si vuole impedire questa trasformazione, è possibile applicare la classe ".not-redraw".
Come si nota, pur se è applicata la classe CSS "icon-style-demo" come per l'icona precedente, CSS non è in grado di modificare il colore dell'immagine.

Se si da un title all'immagine, questo viene riportato nell'SVG trasformato. Se l'immagine non ha title, il sistema lo cerca nel padre diretto e lo applica.
