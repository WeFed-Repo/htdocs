List of Companies

This form displays the companies managed by the licence. The data are automatically loaded by the activation procedure.

The list includes the following data:

Company name: name of the company. The link allows you to access the data to modify them.
SIA: it is the code assigned to the company to be identified in the Rete Nazionale Interbancaria (Interbank Domestic Network). This code is assigned by SIA (Società Italiana per l'Automazione) and it is univocal.
Sender's Signature: it indicates the Signature of the person in charge, which must be present also in the orders files.
