New Client
Permits the creation of a record for a new client.
The entry interface is divided into five sections:
  • Client Data
  • Bank Data
  • Extra Client Data
  • Default Data for Instructions
  • F24 PrintDestinee Data

Company Name/Account Holder: select the name of the company for which to enter a new client record.

Client data Section
The following fields are present:
  • Client Code: indicate the code with which to identify the client
  • Company name / Surname: indicate the name of the client or the surnameifitis a person
  • Name: indicate the christianname of the client ifitis a person.
  • Date of birth: indicate the date of birth of the client
  • Town of birth: indicate the town of birth of the client
  • Provinceof birth: indicate the province of birth of the client
  • Sex: indicate the sex of the client.
  • Fiscal Code: indicate the fiscal code of the client
  • VAT number: indicate the VAT number of the client
  • Type of Address: select the type of address. It can be
    • Complete postaladdress
    • House address
    • Delivery address
    • Residentialaddress
    • Postal correspondenceaddress
    • Post Box address

  • Address: indicate the client'saddress
  • Number: indicate the housenumber
  • Town: indicate the town of residence of the client
  • Prov.: indicate the province of the client
  • Post Code: indicate the Post Code of the client
  • Status: select the client status
  • Type of code: select the type of code. It can be
    • User
    • RegistrationNumber
    • Fiscal Code
    • Client Code(automaticchoice)
    • Supplier Code
    • Commercial portfolio
    • Other

  • Rid Client Code: indicate the code for RID Client
  • Mandate code: indicate the identification code for the document signed by the debtor
  • Subscription date: indicate the date the client signed the mandate
  • Default Instruction: indicate the type of instruction normally associated with the client for collection. It can be
    • M.av
    • Ri.Ba.
    • RID
