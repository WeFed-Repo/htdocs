List of Employees

In this page it is possible to carry out operations on the List of Employees: enter, modify, print and cancel records.

Company Name/Account Holder: select the name of the company whose List of Employeesit is necessary to consult. In the case of the user profile only being able to operate for one company then this field appears filled in.
The information is structured as follows:

  • Surname Name: indicates the name of the employee.
    With a click the employee address record window opens, where it is possible to cancel records with the Cancel button or make changes which will be saved by clicking on the Confirm button.
  • Code: indicates the identification code with which the employee is identifiable
  • Address: indicates the address of the employee
  • Town / Prov.: indicates the Town and Province where the employee is resident
  • Default Instruction: indicates the type of instruction normally associated with the paying of theemployee.It can be
    • Cashier Cheque
    • Non-transferablecashier cheque
    • Salarypayment
    • Telephone top-up
    • To define

The records are ordered according to the Surname Name column.

With the letters of the alphabet displayed above the list, it is possible to filter all the records which start with a certain letter (the letter chosen is highlighted in red).
With the # symbol it is possible to filter those records whose name does not start with a letter (the # symbol is highlighted in red).
With the word ALL it is possible to view the whole list of records available (the world ALL is highlighted in red).


With the Print PDF function, all the records present on the page are displayed in a PDF format (it can be printed or saved to be used later offline).
With the Enter New Employee function it is possible to create a record for a new worker.

Navigation Tools and Common Functions

It is pssible to use the navigation tools and the common functions available: choose the number of records viewed per page.
