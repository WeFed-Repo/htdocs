New Foreign Creditor

Permits the creation of a record for a new foreign creditor. The entry interface consists of one section:

  • Foreign Creditor Data

Company Name/Account Holder: select the name of the company for which to enter a new foreign creditor record.

Foreign creditor data section
The following fields are present:
  • Creditor code: indicate the code with which to identify the foreign creditor
  • SIA: indicate the uniqueidentification code of the foreign creditor
  • Foreigncode
  • Company Name: indicate the name of the foreign creditor
  • Creditor code assigned by the assignor: indicate the code with which the foreign creditor is identifiable by the assignor
  • Address: indicate the foreign creditor's address
  • Post Code: indicate the Post Code of the foreign creditor
  • Town: indicate the town of residence of the foreign creditor
  • Prov.: indicate the province of the foreign creditor
  • Country :select the country of the foreign creditor
  • Creditor signature: indicate the name of the creditor which should appear in the instruction files sent from the position

Click on Enter to confirm the data entered.
Click on Cancel to cancel the operation.
