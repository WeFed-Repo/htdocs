New Supplier

Permits the user to create a record for a new supplier

The entry interface is divided into three sections:

  • Supplier Data
  • Section of Bank Coordinates for Payments
  • Extra Information regarding Supplier Section

Company Name/Account Holder: select the name of the company for which to enter a new supplier record.


Supplier data Section

The following fields are present:
  • Supplier Code: indicate the code with which to identify the supplier
  • CUC: indicate the unique code assigned to the supplier for identification within Customer to Business Interaction (CBI) interactions.
  • Company name: indicate the name of the supplier
  • VAT number: indicate the VAT number of the supplier
  • Fiscal Code: indicate the fiscal code of the supplier
  • Type of address: select the type of address. It can be
    • Complete Postal Address
    • Present Address
    • ShipmentAddress
    • ResidencyAddress
    • Postal CorrespondenceAddress
    • Post Box
  • Address:indicate the address of the supplier
  • Number: indicate the street number
  • Town: indicate the supplier's town
  • Post Code
  • Prov.:indicate the supplier's province
  • Status: select the supplier status
  • Type of code. It can be:
    • User
    • RegistrationNumber
    • Fiscal Code
    • Client code
    • Supplier code (proposedautomatically)
    • Commercial Portfolio
    • Other

Section of Bank Coordinates for Payments

The fields present are as follows:
  • ABI: indicate the ABI code of the bank of the supplier. With the magnifying glass it is possible to look up a code different to the one present in the record.
  • CAB: indicate CAB code of the bank of the supplier. With the Check button it is possible to check the existence of the branch.
  • Account Number: indicate the account number of the bank of the supplier
  • CIN: indicate the check key of the bank coordinates code of the supplier's bank account
  • IBAN: indicate the international coordinates of the supplier's bank account. With the Help IBAN button it is possible to see how it is structured.
  • BIC: (BankIdentifier Code) indicate the international code of the foreignbank
  • BankDescription: indicate the company name of the supplier'sbank
  • Branch location: indicate the location of the supplier'sbankbranch


Extra Information regarding Supplier Section

The section (which is accessed via the arrows present at the end of the heading row) contains optional data useful for SEPA payments. The fields present are:

  • Type of subject:select the type of subject. It can be
    • Physical person (the Type of Code field is filled in automatically with CUC CBI)
    • Legal Representative (the Type of Code field is filled in automatically with Fis.C. / VAT)
  • Type of Code: indicate the type of code. It can be
    • Fiscal Code / VAT
    • BEIidentification
    • CUC CBI
  • Code: indicate the name and numberassigned by the entity for identification
  • Code Description: indicate the entity which assigns the code.

Click on the Enter button to confirm the data entered.

Click on Cancel to cancel the operation.
