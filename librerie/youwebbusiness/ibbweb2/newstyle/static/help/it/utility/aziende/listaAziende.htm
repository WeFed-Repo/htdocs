Elenco Aziende

In questa maschera vengono visualizzate le aziende gestite dalla postazione. I dati sono caricati in automatico dalla procedura di attivazione.

L'elenco riporta, i seguenti dati:

Ragione sociale: denominazione dell'azienda. Il link permette l'accesso ai dati per una eventuale modifica.
SIA: è il codice assegnato all'azienda per la sua identificazione nella Rete Nazionale Interbancaria. Tale codice è assegnato dalla SIA (Società Italiana per l'Automazione) ed è univoco.
Firma Mittente: indica la firma del responsabile, che dovrà risultare nei file di disposizioni.
