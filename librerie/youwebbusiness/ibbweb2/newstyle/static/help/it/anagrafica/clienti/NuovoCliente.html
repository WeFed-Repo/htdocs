Nuovo cliente
Permette di creare l’anagrafica di un nuovo cliente.
La maschera d’inserimento è suddivisa in cinque sezioni:
  • Dati Cliente


  • Dati Banca


  • Sezione Dati Aggiuntivi Cliente


  • Dati Default per Disposizioni


  • Dati Destinatario Stampa F24


Rag. Soc. /Intestatario: selezionare l’azienda per la quale inserire il nuovo cliente.

Sezione Dati Cliente
I campi presenti sono:
  • Codice Cliente: indicare il codice con cui identificare il cliente


  • Ragione sociale / Cognome: indicare il nominativo del cliente oppure il cognome se si tratta di una persona fisica


  • Nome: indicare il nome del cliente se si tratta di una persona fisica


  • Data di nascita: indicare la data di nascita del cliente


  • Comune di nascita: indicare il comune di nascita del cliente


  • Provincia di nascita: indicare la provincia di nascita del cliente


  • Sesso: indicare il sesso del cliente.


  • Codice Fiscale: indicare il codice fiscale del cliente


  • P. IVA: indicare la Partita IVA del cliente


  • Tipo Indirizzo: selezionare il tipo di indirizzo. Può essere:

    • Indirizzo postale completo
    • Indirizzo di domicilio
    • Indirizzo di consegna
    • Indirizzo di residenza
    • Indirizzo di corrispondenza postale
    • Indirizzo della casella postale


  • Indirizzo: indicare l’indirizzo del cliente


  • Numero: indicare il numero civico del cliente


  • Comune: indicare il comune di residenza del debitore


  • Prov.: indicare la provincia del cliente


  • CAP: indicare il Codice di Avviamento Postale


  • Stato: selezionare lo Stato del cliente


  • Tipo codice: selezionare il tipo codice. Può essere:

    • Utenza
    • Matricola
    • Codice Fiscale
    • Codice cliente (proposto in automatico)
    • Codice fornitore
    • Portafoglio commerciale
    • Altri


  • Codice Cliente Rid: indicare il codice cliente Rid


  • Codice mandato: indicare il codice identificativo del mandato sottoscritto dal debitore


  • Data sottoscrizione: indicare la data di sottoscrizione del mandato


  • Disposizione Default: indicare il tipo di disposizione che, di norma,regola i rapporti di incasso con il cliente. Può essere:
    • M.av
    • Ri.Ba.
    • RID


Sezione Dati Banca
I campi presenti sono:
  • Banca del cliente: : indicare la ragione sociale della banca del cliente.


  • ABI: indicare il codice ABI della banca del cliente. Tramite la lente è possibile richiamare un codice diverso da quello presente in rubrica


  • CAB: indicare il codice CAB della banca del cliente. Tramite il pulsante Verifica è possibile accertare l'esistenza dello sportello bancario


  • N. Conto: indicare il numero di conto corrente del cliente


  • CIN: indicare la chiave di controllo delle coordinate bancarie del conto del cliente


  • IBAN: indicare le coordinate internazionali del conto del cliente. Tramite il pulsante Aiuto IBAN è possibile sapere come viene strutturato


  • BIC: (Bank Identifier Code) indicare il codice internazionale della banca estera



Sezione Dati Aggiuntivi Cliente
I campi presenti sono:
  • Tipo Soggetto: selezionare il tipo di soggetto. Può essere:
    • Persona Fisica (il campo Tipo Codice si valorizza in automatico con C.Fis. / P.Iva / VAT)
    • Persona Giuridica (il campo Tipo Codice si valorizza in automatico con CUC CBI)
  • Tipo Codice: selezionare il tipo codice. Può essere:
    • C.Fis. / P.Iva / VAT
    • CUC C.B.I.
    • Identificativo BEI
    • Altro Codice
  • Codice: indicare il nome o il numero assegnato da un'entità per il suo riconoscimento
  • Descrizione Codice: indicare l'entità che assegna il codice.

Sezione Dati Default per Disposizioni
I campi presenti sono:
  • Descrizione Ri.Ba.: selezionare il tipo di documento che deve essere rilasciato al debitore all'esazione del debito. Può essere:
    • Conferma ordine di bonifico
    • Ricevuta bancaria


  • Esiti R.I.D.: indicare se per gli esiti R.I.D. è richiesta la notifica del pagamento. Può essere:
    • È richiesta notifica del pagato
    • Non è richiesta notifica del pagato
    • Non specificato


  • Esiti Ri.Ba.: indicare se per gli esiti Ri.Ba. è richiesta la notifica del pagamento. Può essere:
    • È richiesta notifica del pagato
    • Non è richiesta notifica del pagato
    • Non specificato


  • Tipo Incasso: selezionare la tipologia di RID. Può essere:
    • Commerciale
    • Utenza
    • Veloce


  • Stato avviso Ri.Ba.: indicare l’incaricato della stampa e dell’invio dell’avviso di pagamento. Può essere:
    • Avviso a cura della banca
    • Non specificato



Sezione Dati Destinatario Stampa F24
La compilazione di questa sezione permette di valorizzare in automatico i dati del Destinatario di Stampa nel Pagamento F24. I campi presenti sono:
  • Quietanza F24: indicare il destinatario della stampa dell'attestazione di pagamento. Può essere:
    • Invio attestazione destinatario stampa
    • Invio attestazione titolare C/C addebito


  • Ragione Sociale: inserire la Ragione Sociale del destinatario di stampa


  • Indirizzo: inserire l’indirizzo di domicilio del destinatario di stampa


  • Comune: indicare il comune di domicilio del destinatario di stampa


  • CAP: inserire il CAP del comune di domicilio del destinatario di stampa


  • Prov.: inserire la sigla della provincia del domicilio del destinatario di stampa.


Cliccare il pulsante Inserisci per confermare l’inserimento dei dati.

Cliccare il pulsante Annulla per annullare l’operazione.
