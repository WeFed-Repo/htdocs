List of Foreign Creditors

In this page it is possible to carry out operations on the Foreign Creditors list: enter, modify, print and cancel records.

Company Name/Account Holder: select the name of the company whose Foreign Creditors list it is necessary to consult. In the case of the user profile only being able to operate for one company then this field appears filled in.

The information is structured as follows:

  • Company Name / Surname Name: indicates the name of the foreign creditor.
    With a click the Foreign Creditor window opens, where it is possible to cancel records with the Cancel button or make changes which will be saved by clicking on the Confirm button.
  • Code: indicates the identification code with which the foreign creditor is identifiable
  • Address: indicates the address of the foreign creditor
  • Town / Prov.: indicates the Town and Province where the foreign creditor is resident

The records are ordered according to the Company Namecolumn.
With the letters of the alphabet displayed above the list, it is possible to filter all the records which start with a certain letter (the letter chosen is highlighted in red).
With the # symbol it is possible to filter those records whose name does not start with a letter (the # symbol is highlighted in red).
With the word ALL it is possible to view the whole list of records available (the world ALL is highlighted in red).

With the Print PDF function, all the records present on the page are displayed in a PDF format (it can be printed or saved to be used later offline).
With the Enter New Foreign Creditor function it is possible to create a record for a new foreign creditor.

Navigation Tools and Common Functions

It is possible to use the navigation tools and the common functions available: choose the number of records viewed per page.
