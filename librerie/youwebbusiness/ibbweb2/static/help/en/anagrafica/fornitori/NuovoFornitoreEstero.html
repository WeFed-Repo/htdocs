New Foreign Supplier

Permits the user to create a record for a new foreign supplier

The entry interface is divided into three sections:

  • Supplier Data
  • Bank Coordinates for Foreign Payments
  • (Default) Data for Payments

Company Name/Account Holder: select the name of the company for which to enter a new foreign supplier record.

SupplierData Section
The following fields are present:

  • Supplier Code: indicate the code with which to identify the supplier
  • Company name: indicate the name of the supplier
  • Fiscal Code: indicate the fiscal code of the supplier
  • VAT number: indicate the VAT number of the supplier
  • Address:indicate the address of the supplier
  • Town: indicate the supplier's town
  • Post Code
  • Prov.:indicate the supplier's province
  • Country.

Bank Coordinates for Foreign Payments
The fields present are as follows:

  • Country Code: select the overseas country of the supplier
  • IBAN: indicate the international coordinates of the supplier's bank account. With the Help IBAN button it is possible to see how it is structured.
  • BIC/SWIFT: (Bank Identifing Code) indicate the international code of the foreign bank
  • Bank Description: indicate the company name of the supplier's bank
  • Branch location: indicate the location of the supplier's bank branch

(Default) Data for Payments
The fields present are as follows:

  • Debit charges: indicate who must pay the bank charges. It can be:/li>
    • Beneficiary
    • Eachonehisown
    • Debtor
  • - Type of payment: select the paymenttype to use to regulate the operations. It can be:
    • Accredit C/A
    • Creditor Bank Cheque
    • OrdererBank Cheque
    • Kept c/o bank
  • Currency of accredit account: indicate the currency for accrediting the amount to the beneficiary. With the magnifying glassit is possible to look up a code different to the one present in the records.
  • Type of notice: select the notification method for the foreign supplier upon payment. It can be
    • Telephone
    • Telex
  • Type of contact: select the type of contact. It can be
    • Contact for the bank
    • Contact for the notification of third parties
  • Type of beneficiary date: select which date to assign to foreign supplier transactions.It can be
    • Availability date requested
    • Accreditedvalue date

Click on the Enter button to confirm the data entered.

Click on Cancel to cancel the operation.
