Elenco clienti

In this page itispossible to carry out operations on the Client list: enter, modify, print and cancelrecords.

Company Name/Account Holder: select the name of the company whose Client listit is necessary to consult. In the case of the user profile only being able to operate for one company then this field appears filled in. The information is structured as follows:

Rag. Soc./Intestatario: selezionare il nominativo della società per la quale si desidera visualizzare la rubrica Clienti. Nel caso in cui la profilatura del proprio utente consenta l’operatività su un’unica società, il campo risulta precompilato con la relativa Ragione Sociale/Intestazione.

Le informazioni visualizzate sono così strutturate:
  • Company Name / Surname Name: indicates the name (Company name if it is a company or Surname and Name if it is a person) of the client.
    With a click the Records window of client addresses opens, where it is possible to cancel records with the Cancel button or make changes which will be saved by clicking on the Confirm button.
  • Client Code: indicates the identification code with which the client is identifiable
  • Address: indicates the address of the client
  • Town / Prov.: indicates the Town and Province where the client is resident
  • Defaul Instruction: indicates the type of instruction normally associated with the client for collections.It can be
    • M.av.
    • Ri.Ba.
    • RID


The records are orderedaccording to the Company Name/SurnameNamecolumn.
With the letters of the alphabet displayed above the list, it is possible to filter all the records which start with a certain letter (the letter chosen is highlighted in red).
With the # symbol it is possible to filter those records whose name does not start with a letter (the # symbol is highlighted in red).
With the word ALL it is possible to view the whole list of records available (the world ALL is highlighted in red).

With the Print PDF function, all the records present on the page are displayed in a PDF format (it can be printed or saved to be used later offline).
With the Enter New Client function it is possible to create a record for a new client.

Navigation Tools and Common Functions

It is possible to use the navigation tools and the common functions available: choose the number of records viewed per page.
