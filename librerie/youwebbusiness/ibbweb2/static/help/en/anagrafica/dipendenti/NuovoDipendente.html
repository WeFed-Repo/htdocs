New Employee
Permits the creation of a record for a new employee.
The entry interface is divided into two sections:
  • Employee Data
  • Bank Data

Company Name/Account Holder: select the name of the company for which to enter a new employee record.

Employee Data Section

The following fields are present:

  • Employee Code: indicate the code with which the employeeisidentifiable
  • SurnameName: indicate the surname and name of theemployee
  • Fiscal Code: indicate the fiscal code of the employee
  • Address: indicate the address of the employee
  • Town: indicate the town of residence of the employee
  • Prov.: indicate the province where the employee lives
  • Post Code: indicate the Post Code of the employee
  • Type of code: select the type of code. It can be
    • User
    • RegistrationNumber (proposedautomatically)
    • Fiscal Code
    • Client code
    • Supplier code
    • Commercial Portfolio
    • Other
  • Default Instruction: indicate the type of instructionnormallyused to pay the empoloyee. It can be:
    • Cashier Cheque
    • Non-transferablecashier cheque
    • Salarypayment
    • Telephone top-up
    • To define
  • Cause Code: indicate the cause code. It can be:
    • Payments - Generic
    • Payments - pension
    • Payments - salaries (proposedautomatically)
    • To define

Bank Data Section
The fields present are as follows:

  • Bank of the employee: indicate the name of the bank of the employee
  • ABI: indicate the ABI code of the bank of the employee. With the magnifying glass it is possible to look up a code different to the one present in the record.
  • CAB: indicate CAB code of the bank of the employee. With the Check button it is possible to check the existence of the branch.
  • Account Number: indicate the account number of the bank of the employee
  • CIN: indicate the check key of the bank coordinates code of the employee's bank account
  • IBAN: indicate the international coordinates of the employee's bank account. With the Help IBAN button it is possible to see how it is structured.
  • Telephone Company: select the telephone company of the telephone number to do top-ups for. It can be
    • Vodafone
    • Tim
    • Wind
    • Tiscali
    • H3G


  • Telephone Number: indicate the telephone number for telephone top-ups.

Click on the Enter button to confirm the data entered.

Click on Cancel to cancel the operation.
